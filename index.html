Crucipixel

Crucipixel is a simple application for Linux to play with crucipixels (also called nonograms).

Easy to use

If you have a modern Linux environment, chances are you won't need to do anything to play Crucipixel! It simply requires Python3 and PyGObject to be run.

Simply run crucipixel and enjoy!

Install schemes

If you want to add some basic scheme examples, just run setup.sh

Add new schemes

You can easily create new schemes with Crucipixel's internal editor; if you prefer to code things by hand, the schemes are encoded in a simple JSON format!
